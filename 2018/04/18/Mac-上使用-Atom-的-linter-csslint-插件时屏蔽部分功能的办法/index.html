Mac 上使用 Atom 的 linter-csslint 插件时屏蔽部分功能的办法

Posted on 2018-04-18 Disqus:

我在 Atom 中使用了 linter-csslint 插件，对这个插件的大部分功能都很满意，但是也遭遇小坑一个：它提示我“Rule doesn’t have all its properties in alphabetical order.”

意思就是我 CSS 中某处规则没按字母顺序排序，但是我不想按字母排序啊，我觉得字母排序有点别扭，找了半天没找设置在哪，去项目的 GitHub 上面去看，作者说可以在项目中新建一个叫做“.csslintrc”的全局文件，在这个文件中把这个功能声明为“false”就不会有提示了。

这太繁琐啦，我这暴脾气，直接改源代码吧。一番好找，找到了目标文件：/Users/你的用户名/.atom/packages/linter-csslint/node_modules/csslint/dist/csslint-node.js

直接在文件里面搜索那句警告——“Rule doesn’t have all its properties in alphabetical order”，把这个 Rule 对应的方法直接注释掉，保存，大功告成。
