Spring MVC 框架简介

Spring MVC 中的重要组件

  1. DispatcherServlet：前端控制器，接收所有请求（如果路径配置为“/”，则不包括 JSP 文件）
  2. HandlerMapping：解析请求格式的，判断希望要执行哪个具体的方法
  3. HandlerAdapter：负责调用具体的方法
  4. ViewResolver：视图解析器，解析结果，负责准备跳转到具体的物理视图
